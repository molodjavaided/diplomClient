Интернет-магазин украшений с административной панелью для администратора, корзиной покупок и системой авторизации.

Функции:
Для пользователя:

- Поиск товаров
- Категории товаров
- Сортировка товаров по стоимости
- Страница товара с информацией:
- Добавление товара в корзину
- Корзина покупок:
- Отображение общей стоимости товаров в корзине
- Отображение количества товара в корзине
- Удаление товаров из корзины
- Изменение количества товара в корзине
- Оформление заказа
- Очищение товаров из корзины
- Регистрация и вход

Для администратора:

- Добавление новых товаров
- Изменение текущих товаров
- Удаление товаров
- Добавление категорий товаров
